Tideway — Linux install

Tideway ships as a Flatpak for Linux. Two ways to install.

Auto-updating (recommended)

Subscribe to this Flatpak repo once; flatpak update pulls new releases.

flatpak remote-add --user --if-not-exists tideway \
    https://j-m-punk.github.io/tideway/tideway.flatpakrepo
  flatpak install --user tideway com.tidaldownloader.Tideway

One-shot bundle

Download Tideway-1.10.0.flatpak from the latest release, then:

flatpak install --user --bundle Tideway-1.10.0.flatpak

Won't auto-update; grab a newer bundle when you want to upgrade.
