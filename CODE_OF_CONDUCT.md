# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1][cc] as its
code of conduct. Read the full text at the link below. By participating
in this project (filing issues, opening pull requests, joining
discussions) you agree to abide by it.

[Read the full Contributor Covenant 2.1][cc]

[cc]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

If you experience or witness behavior that violates the Contributor
Covenant in any Tideway space (issues, pull requests, discussions),
report it to the maintainer at:

**william.pratt.three@gmail.com**

Reports are kept confidential. The maintainer will review every
report, decide on appropriate action, and respond to the reporter.

## Scope

This Code of Conduct applies to all project spaces, including the
GitHub repository, issue tracker, pull request reviews, and any
related communication channels.
